# UnityWorkbox 

[简体中文](./README-ZH-cn.md)

Unity3d developer workbox that has some useful Unity3d script extensions and editor tools

## UI Management

* UIManager
* PanelUI
* Group/Exclusive/Floating

## Network

* Netwoker
* Connector
* MessageHandler

## Tools
* Git
* ConfigurableArray
* ConsoleLogger
* JsonList & JsonDict
